# Advice Generator App

## Overview

The **Advice Generator App** is a simple web application that displays random pieces of advice. It features a clean and responsive design, utilizing HTML and CSS.

## Features

- Display advice with a dynamic design.
- Responsive layout for a seamless experience on various devices.
- Customized font styles and color scheme.

## Usage

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/advice-generator-app.git
   ```

2. Open `index.html` in your preferred web browser.

## Technologies Used

- HTML
- CSS
- Manrope Font (Imported from Google Fonts)

## Folder Structure

- `images`: Contains favicon, patterns, and icons used in the application.
- `style.css`: Stylesheet for the HTML file.
- `index.html`: Main HTML file for the Advice Generator App.
